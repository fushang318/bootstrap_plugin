# Overlay-Bootstrap

鼠标悬浮的内容上动态浮动出遮盖层（纯css实现）


## 前段代码

HTML
```html
    <div class="panel-body">
        <p>例子</p>
        <div>鼠标悬浮在我身上，会有DIV滑出覆盖住我

        <div class="panel-primary-overlay-right">
            <p>我是覆盖</p>
        </div>
    </div>
```

根据 css 定义滑出的方向，另外也有一些其他定义参数（以标签属性的方式设置）
