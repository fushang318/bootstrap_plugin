# tooltip
Bootstrap 工具提示（Tooltip）插件 当您想要描述一个链接的时候，工具提示（Tooltip）就显得非常有用。

## 网址 
http://getbootstrap.com/javascript/#tooltips

## 前段代码

HTML
```html
  <a href="#" id="demo1" title="Example tooltip">请悬停在我的上面</a>
```

JS
```js
 $('#demo1').tooltip(各种参数)
```
